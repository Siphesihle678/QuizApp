üî¢ Basic Mathematical Functions

SUM Function

Purpose: Adds up all the numbers in a specified range of cells.

=SUM(number1, [number2], ...)
Example:

=SUM(A1:A10) - Adds all values from cell A1 to A10

=SUM(5, 10, 15) - Adds the numbers 5, 10, and 15

üí° Tip:

You can also use =SUM(A1:A10, B1:B5) to add multiple ranges at once.

AVERAGE Function

Purpose: Calculates the arithmetic mean of a group of numbers.

=AVERAGE(number1, [number2], ...)
Example:

=AVERAGE(D2:F2) - Calculates the average of cells D2, E2, and F2

=AVERAGE(A1:A20) - Calculates the average of 20 cells

üí° Tip:

The AVERAGE function automatically ignores text and empty cells in the range.

MAX Function

Purpose: Returns the largest value in a set of values.

=MAX(number1, [number2], ...)
Example:

=MAX(A1:A50) - Finds the highest score in column A

=MAX(10, 25, 5, 30) - Returns 30

MIN Function

Purpose: Returns the smallest value in a set of values.

=MIN(number1, [number2], ...)
Example:

=MIN(B1:B100) - Finds the lowest value in column B

ü§î Logical Functions

IF Function

Purpose: Performs a logical test and returns one value if TRUE, another if FALSE.

=IF(logical_test, value_if_true, value_if_false)
Example:

=IF(H2>=50, "Pass", "Fail") - Shows "Pass" if score is 50 or more, "Fail" otherwise

=IF(A1>0, A1*2, 0) - Doubles the value if positive, returns 0 if not

üí° Tip:

You can nest IF functions for multiple conditions: =IF(A1>80, "A", IF(A1>60, "B", "C"))

Nested IF Functions

Purpose: Multiple IF statements inside each other to test several conditions.

=IF(condition1, result1, IF(condition2, result2, result3))
Example:
=IF(score>=90, "A+", IF(score>=80, "A", IF(score>=70, "B", IF(score>=60, "C", "F"))))
‚ö†Ô∏è Warning:

Too many nested IFs can make formulas hard to read. Consider using VLOOKUP or IFS (Excel 2019+) for complex scenarios.

üìä Counting Functions

COUNTIF Function

Purpose: Counts the number of cells that meet a specific condition.

=COUNTIF(range, criteria)
Examples:

=COUNTIF(A1:A10, ">50") - Counts cells with values greater than 50

=COUNTIF(B1:B20, "Pass") - Counts cells containing "Pass"

=COUNTIF(C1:C100, "*") - Counts non-empty cells

üí° Tip:

Use wildcards: * for any characters, ? for single character.

COUNT Function

Purpose: Counts the number of cells that contain numbers.

=COUNT(value1, [value2], ...)
Example:

=COUNT(A1:A100) - Counts how many cells contain numbers

üìù Text Functions

CONCATENATE Function

Purpose: Combines text from two or more cells into one cell.

=CONCATENATE(text1, [text2], ...)
Example:

=CONCATENATE(A1, " ", B1) - Combines first name and last name with a space

=CONCATENATE("Grade: ", C1) - Adds "Grade: " before a value

üí° Tip:

In newer Excel versions, you can use the & operator: =A1&" "&B1

üîç Lookup Functions

VLOOKUP Function

Purpose: Searches for a value in the first column of a table and returns a value in the same row from another column.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
Example:

=VLOOKUP("John", A1:C10, 2, FALSE) - Looks for "John" in column A and returns the value from column B

‚ö†Ô∏è Important:

VLOOKUP searches from left to right only. The lookup value must be in the leftmost column of the table.

üé® Conditional Formatting

Highlighting Values

Purpose: Automatically format cells based on their values or conditions.

Steps to highlight scores below 40:
  1. Select the cells you want to format
  2. Go to Home ‚Üí Conditional Formatting
  3. Choose New Rule
  4. Select Format only cells that contain
  5. Set condition: Cell Value less than 40
  6. Choose formatting (e.g., red background)
  7. Click OK
üí° Tip:

You can use formulas in conditional formatting for more complex conditions.

‚ùå Common Mistakes to Avoid

1. Forgetting Equal Sign

Wrong: SUM(A1:A10)
Correct: =SUM(A1:A10)

2. Incorrect Cell References

Wrong: =SUM(A1:A10, B1:B10) (missing comma)
Correct: =SUM(A1:A10,B1:B10)

3. Text vs Numbers

Make sure numbers are actually stored as numbers, not text. Use =VALUE() to convert text to numbers.

4. VLOOKUP Range Issues

Always ensure your lookup value is in the leftmost column of your table range.
